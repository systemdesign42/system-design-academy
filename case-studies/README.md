# Case Studies

Real-world system design examples from leading technology companies. Learn how they solved scaling challenges, handled failures, and built systems serving billions of users.

## By Theme

| Category | Companies | Key Concepts |
|----------|-----------|--------------|
| [Scaling](./scaling/) | Instagram, LinkedIn, Netflix, Disney+ | Horizontal scaling, microservices, CDN |
| [Real-Time](./real-time/) | Slack, WhatsApp, Canva, Disney+ Hotstar | WebSockets, pub/sub, presence |
| [Databases](./databases/) | YouTube, Quora, Tumblr, Cloudflare | Sharding, replication, migration |
| [Infrastructure](./infrastructure/) | AWS, Nginx, Cloudflare, Facebook | Load balancing, serverless, storage |
| [Payments](./payments/) | Stripe, PayPal, Razorpay | Idempotency, rate limiting, transactions |
| [Search & Discovery](./search-discovery/) | Google, Uber, Tinder | Geospatial, indexing, matching |
